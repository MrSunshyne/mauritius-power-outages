# Power Outages - Mauritius

A web app that shows Power Outage information for Mauritius using data from [MrSunshyne/mauritius-dataset-electricity](https://github.com/MrSunshyne/mauritius-dataset-electricity).

## Features

- View today's and tomorrow's scheduled power outages
- Navigate through historical outage data
- Statistics dashboard with charts showing:
  - Outages by day of week
  - Outages by month
  - Outages by district
  - Outages by hour of day

## Tech Stack

- Nuxt 3
- Vue 3
- Tailwind CSS v4
- ApexCharts
- VueUse
- date-fns

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Generate static site
pnpm generate
```
